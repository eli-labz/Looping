# Physical-World Boundaries

Looping is physical-world aware, human-supervised, and simulator-friendly first.

## Staged Capability Model

1. Text tasks: summarize, classify, draft, compare, extract.
2. Digital human-action tasks: browser, spreadsheet, email, dashboards.
3. Physical-world tasks: observe, recommend, simulate, route to human, document action.

## Non-Goals (Current Stage)

- Unattended robot autonomy
- Irreversible actuation without human approval
- Hidden execution without trace logging
