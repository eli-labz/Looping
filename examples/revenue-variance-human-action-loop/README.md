# Revenue Variance Human-Action Loop

This example demonstrates a supervised, auditable workflow for revenue variance analysis.

## Flow

1. Open dashboard.
2. Export CSV.
3. Open spreadsheet.
4. Analyze variance.
5. Draft email.
6. Ask human before sending.
7. Log every observation, action, outcome, and approval.

## Why This Milestone

This sequence exercises browser, spreadsheet, and email task families with explicit verification and governance controls.

## Included Files

- `workflow.yaml`: canonical action sequence.
- `sample-action-ledger.jsonl`: sample tokenized trace.
